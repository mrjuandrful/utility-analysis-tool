# Typeform Quick Start - 10 Minute Setup

## The Fast Way (Recommended)

### Step 1: Go to Typeform (2 minutes)
```
1. Open: https://typeform.com
2. Click "Sign Up" (or log in if you have account)
3. Create account or log in
4. Click "Create" → "New form"
5. Choose "Blank form"
6. Name it: "Utility Bill Analysis"
```

### Step 2: Add Each Question (One at a time)

#### Question 1: Welcome Screen
```
Type: Click "Cover" (at bottom left)
Title: "Welcome to Utility Analysis!"
Description: "We'll analyze your utility bills to show you where your money goes."
Button: "Start" (already default)
```
✅ Done. Click outside to save. Click "+" to add next question.

---

#### Question 2: Name
```
Type: "Short text"
Question: "What's your full name?"
Required: Yes
```
✅ Click "+" for next question.

#### Question 3: Email
```
Type: "Email"
Question: "What's your email address?"
Description: "We'll send your analysis here"
Required: Yes
```
✅ Click "+" for next question.

#### Question 4: Address
```
Type: "Short text"
Question: "What's your street address?"
Description: "(For location context)"
Required: Yes
```

#### Question 5: Utilities You Have
```
Type: "Multiple choice"
Question: "Which utilities do you have?"
Description: "Select all that apply"

Add these options:
□ Electricity only
□ Natural Gas only
□ Electricity + Gas
□ Electricity + Gas + Water
□ Other (please specify)

Multiple answers: Toggle ON (allow multiple selections)
Required: Yes
```
✅ Click "+" for next question.

---

#### Question 6: Utility Companies
```
Type: "Long text"
Question: "Which utility companies serve you?"
Description: "List the names and service types. Example: PSEG (electric/gas), Veolia (water)"
Required: Yes
```
✅ Click "+" for next question.

---

#### Question 7: Upload Utility Bills
```
Type: "File upload"
Question: "Please upload your utility bill PDFs"
Description: "We need 3-6 months of recent bills. You can upload multiple files."

Settings:
- File types: PDF only (uncheck others)
- Multiple files: Toggle ON
- Max file size: 25MB per file

Required: Yes
```
✅ Click "+" for next question.

---

#### Question 8: What Interests You?
```
Type: "Multiple choice"
Question: "What are you most interested in understanding?"
Description: "Select your top 3 priorities"

Add these options:
□ Why my bill is so high
□ Monthly cost breakdown
□ Which appliances use most energy
□ Seasonal trends (winter vs summer)
□ Savings recommendations
□ Payment options/billing credits
□ Comparison to average customers
□ Solar/renewable energy options

Multiple answers: Toggle ON
Required: Yes
```
✅ Click "+" for next question.
